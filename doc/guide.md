# 用户

- 发消息时收件人根据输入即时提示
- 点按消息区域头部可折叠/展开消息内容，同时按Shift可折叠/展开全部
- 发消息：输入时按Ctrl+回车可即时发送

用户（员工）也是一个信息，在企业内也会形成信息流。
入职：建立帐号
离职：停用帐号，交接工作

帐号创建过程
- 创建，初始密码
- 通知到员工（系统外方式，系统通知）
- 员工登录，设置密码
